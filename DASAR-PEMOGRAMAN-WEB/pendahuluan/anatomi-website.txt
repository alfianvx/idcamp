Pondasi sebuah Website:
1.  HTML        = Markup Language yang menentukan struktur atau kerangka website.
2.  CSS         = Declarative Language yang bergfungsi untuk memberikan styling pada website.
3.  JavaScript  = Bahasa Pemograman yang di dukung oleh browser untuk memberikan interaktivitas pada Halaman Web.

Jika di analogikan dengan Hewan, maka :
1.  HTML adalah skeleton / tulang nya.
2.  CSS adalah kulit yang menutupi tulang.
3.  JavaScript adalah otak yang memberikan arahan supaya hewan bisa beraktifitas.
