Pada dasarnya dokumen HTML memerlukan struktur untuk menampilkan halaman pada web dengan baik.

Ketika kita ingin menuliskan kode HTML, maka kita harus mendefinisikan sebuah element bernama DOCTYPE
di ikuti dengan tipe / nama dokumen yang akan kita tulis.
Contoh jika kita ingin menuliskan kode HTML : <!DOCTYPE html>

Setelah itu, kita dapat menuliskan element <html></html> sebagai pembungkus utama untuk child element nya,
seperti element head, body, ataupun footer.

Dokument pada HTML membentuk sebuah hierarki yang di sebut sebagai DOM TREE.

Kita juga dapat memberikan komentar (element yang di tulis di body tetapi tidak akan di tampilkan di halaman web)
Contoh: <!-- ini komentar -->
